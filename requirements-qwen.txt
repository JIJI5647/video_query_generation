# Server-only dependencies for the qwen3_omni caption backend.
#
# Heavy + require a GPU. NOT needed for the default Gemini pipeline, the local
# lightweight checks, or tests — the backend imports them lazily (only on the
# first real inference call). Install ONLY on the GPU inference server:
#
#     pip install -r requirements.txt -r requirements-qwen.txt
#
# Versions are intentionally unpinned: Transformers must be recent enough to
# support Qwen3-Omni (Qwen3OmniMoeForConditionalGeneration + Qwen3OmniMoeProcessor),
# and torch must be a build matching the GPU driver's CUDA. Follow the official
# Qwen3-Omni instructions if these floating versions don't resolve.

# --- Shared by both engines ---
transformers          # Qwen3-Omni model + processor
qwen-omni-utils       # process_mm_info (video/audio preprocessing)
torch                 # install the CUDA build that matches your GPU driver
accelerate            # device_map="auto" sharding for the transformers engine

# --- Engine: vllm (default, --qwen-engine vllm) ---
# Fast, but the vLLM build must match BOTH the driver's CUDA and Qwen3-Omni's
# multimodal support. If vLLM won't load the model as multimodal on your
# CUDA/driver, use --qwen-engine transformers instead (no vLLM needed).
vllm

# --- Optional: faster attention for the transformers engine ---
# Enable with --qwen-attn-impl flash_attention_2 once installed.
# flash-attn
