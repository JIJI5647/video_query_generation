DIMENSIONS:

1. relevance_pass
Mark relevance_pass = true if the query is about one of the eight target emotions
— angry, excited, fear, sad, surprised, frustrated, happy, disappointed — or about
observable evidence for one of them. Synonyms are allowed only when they clearly
refer to one of the eight labels.
Mark relevance_pass = false if the query is mainly about a non-target emotion, a
broad mental state, an intention, a thought, a memory, a relationship, or a
plot-level interpretation rather than one of the eight target emotions or its
observable evidence. Do not stretch broad states into the eight target emotions
unless the query and the provided clip(s) clearly support one of the eight labels.

2. answerability_pass
Mark answerability_pass = true only if the provided clip segment(s) contain enough
visible or audible evidence to answer the query. Valid evidence may include facial
expression, gaze, body movement, gesture, posture, interaction, non-verbal audio,
vocal tone, crying, laughing, shouting, hesitation, silence, or an explicit verbal
statement of feeling.
Mark answerability_pass = false if the query can only be answered using plot
context, outside knowledge, character identity, events outside the provided
clip(s), or an unobservable internal state. If the query may be answerable
somewhere else in the full video but not in the provided clip(s), mark
answerability_pass = false.

3. query_quality_pass
Mark query_quality_pass = true if the query is a single clear English question,
self-contained, specific enough for annotation, based on observable visual or
audible evidence, and contains no timestamps, segment IDs, person IDs, character
names, actor names, movie titles, or dataset-specific identifiers.
Mark query_quality_pass = false if the query is vague, too broad, grammatically
unclear, asks multiple questions, includes timestamps or segment IDs, uses
character names, actor names, movie titles, dataset-specific identifiers, or
depends on unobservable mental content.

SUGGESTED REVISION:
Provide suggested_revision only when relevance_pass = true, answerability_pass =
true, and query_quality_pass = false. The revision must only fix wording: do not
change the target person, target emotion, observable evidence, or temporal moment.
If relevance_pass = false or answerability_pass = false, leave suggested_revision
empty.
